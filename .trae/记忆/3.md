## 2. GameMaster Agent 核心循环 (步骤 2.8)

**文件**: `src/agent/game_master.py`

### 核心类

```python
class GameMaster:
    def __init__(self, world_id, player_id, llm_client, db_path)
    def process(self, user_input: str) -> str          # 同步处理
    def process_stream(self, user_input: str) -> Iterator[str]  # 流式输出
```

### 推理循环流程

```
1. 接收玩家输入
   ↓
2. 记录到历史 + 数据库
   ↓
3. 构建上下文 (System Prompt + 裁剪后的历史)
   ↓
4. 调用 DeepSeek (带 tools)
   ↓
5. 检查 tool_calls
   ├─ 有 → 执行工具 → 结果加入历史 → 回到步骤4 (最多10轮)
   └─ 无 → 返回叙事文本
   ↓
6. 保存回复到历史 + 数据库
   ↓
7. 返回给玩家
```

### 关键实现细节

- **工具调用处理**: 需要将 assistant 消息（含 tool_calls）和 tool 消息按顺序加入历史
- **DeepSeek 兼容**: 需要保留 `reasoning_content` 字段
- **最大工具轮次**: `MAX_TOOL_ROUNDS = 10`
- **对话历史加载**: 从 `game_messages` 表加载最近50条

## 6. 对话历史持久化 (步骤 2.12)

**数据库表**: `game_messages`

```sql
CREATE TABLE game_messages (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    world_id INTEGER NOT NULL,
    role TEXT NOT NULL,      -- system/user/assistant/tool
    content TEXT NOT NULL,
    timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (world_id) REFERENCES worlds(id)
);
```

**操作**:
- 加载: `GameMaster._load_history()` - 加载最近50条
- 保存: `GameMaster._save_message()` - 每条消息实时保存
