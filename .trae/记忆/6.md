## 1. Prompt 优化 (步骤 5.1-5.2)

### 1.1 基于调优日志迭代 Prompt (5.1)

**文件**: `src/prompts/gm_system.py`

**优化内容**:

| 优化项 | 修改内容 |
|--------|----------|
| 防止替玩家做决定 | 细化规则1，添加3条子规则，强调"绝对不要替玩家行动、说话、选择" |
| 提升叙事长度 | "每次场景描述不少于50字" → "每次回复至少3-5句话" |
| 增强感官描写 | 添加"使用生动的形容词和比喻"、"战斗要有动作感和紧张感" |
| 风格一致性 | 添加规则7: "保持角色 — 不要跳出GM角色或使用现代网络用语" |
| 强化记忆 | 规则3添加"正确引用玩家之前的行动" |

**调优日志**: `docs/prompt-tuning-log.md`

## 2. 性能优化 (步骤 5.3-5.8)

**文件**: `src/services/context_manager.py`

**核心函数**:

```python
def estimate_tokens(text: str) -> int
# 中文约1.5字/token，英文约4字符/token

def compress_history(history: list[dict], llm: LLMClient | None = None) -> list[dict]
# 策略:
# 1. 保留 System Prompt
# 2. 保留最近 10 轮完整对话
# 3. 将更早的对话用 LLM 做摘要
# 4. 摘要作为一条 assistant 消息插入

def trim_history(history: list[dict], max_tokens: int = MAX_CONTEXT_TOKENS) -> list[dict]
# 从最早的非 system 消息开始删除
```

**触发条件**: `SUMMARY_TRIGGER_TOKENS = 60000`

### 2.2 关键信息提取 (5.4)

**文件**: `src/services/info_extractor.py`

**提取字段**:
- `new_locations`: 新发现的地点
- `new_npcs`: 新遇到的NPC
- `items_obtained`: 获得的物品
- `key_choices`: 玩家做出的重要选择
- `current_objective`: 当前目标

**函数**:
```python
def extract_key_info(history: list[dict], llm: LLMClient | None = None) -> dict
def build_world_summary(world_id: int, history: list[dict], db_path: str | None = None) -> str
```

### 2.3 响应缓存 (5.5)

**文件**: `src/services/cache.py`

**类**: `LRUCache`

| 参数 | 默认值 | 说明 |
|------|--------|------|
| max_size | 100 | 最大缓存条目数 |
| ttl_seconds | 300 | 缓存过期时间 |

**全局实例**: `llm_cache = LRUCache(max_size=200, ttl_seconds=600)`

**方法**:
- `get(prompt, **kwargs)` - 获取缓存
- `set(prompt, response, **kwargs)` - 设置缓存
- `invalidate(prompt_prefix="")` - 清除缓存

### 2.4 预生成 (5.6)

**文件**: `src/services/pregenerator.py`

**函数**:

```python
async def pregenerate_location_description(location_name: str, location_type: str = "普通地点") -> str
async def pregenerate_npc_greeting(npc_name: str, personality: str = "友好") -> str
async def pregenerate_for_location(location_name: str, npc_names: list[str] | None = None)
```

**缓存键前缀**: `PREGEN_PREFIX = "pregen:"`

### 2.6 多模型路由 (5.8)

**文件**: `src/services/model_router.py`

**模型配置**:

| 模型 | 用途 | 关键词触发 |
|------|------|-----------|
| deepseek-chat | 日常对话，速度快 | 默认 |
| deepseek-reasoner | 关键剧情，质量高 | 战斗、boss、决战、死亡、结局、选择、命运、重要、秘密、真相、最终、关键 |

**路由规则**:
- 包含关键词 → deepseek-reasoner
- 对话超过20轮 → deepseek-reasoner
- 其他 → deepseek-chat
