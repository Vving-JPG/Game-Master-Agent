## Step 2.1: runtime_panel 添加 closeEvent 清理 timer

**问题**: BUG-031 — `RuntimePanel` 的 QTimer 在面板关闭时未停止。

**文件**: `2workbench/presentation/ops/debugger/runtime_panel.py`

**修改方案**：
```python
# 在 RuntimePanel 类中添加 closeEvent 方法
def closeEvent(self, event) -> None:
    if hasattr(self, '_refresh_timer'):
        self._refresh_timer.stop()
    super().closeEvent(event)
```

**验收标准**:
- [ ] 关闭调试面板后 timer 停止运行

## Step 2.2: project_selector 字体使用主题变量

**问题**: BUG-029 — `ProjectItemDelegate.paint` 和 Header 中硬编码 `Microsoft YaHei`。

**文件**: `2workbench/presentation/dialogs/project_selector.py`

**修改方案**：
```python
# 在 paint 方法中（L85/93/100），将:
painter.setFont(QFont("Microsoft YaHei", 11, QFont.Weight.Medium))
# 改为:
font_family = p.get("font_family", '"Microsoft YaHei", sans-serif')
painter.setFont(QFont(font_family.split(",")[0].strip('"'), 11, QFont.Weight.Medium))

# L159 Header 中同理:
font_family = p.get("font_family", '"Microsoft YaHei", sans-serif')
title_label.setFont(QFont(font_family.split(",")[0].strip('"'), 14, QFont.Weight.Bold))
```

**验收标准**:
- [ ] 主题切换后字体正确更新
