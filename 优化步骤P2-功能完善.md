## Step 2.1: 修复内嵌服务器线程无法停止

**问题**: BUG-021 — `_stop_service` 只处理了 `subprocess` 模式，没有处理 `threading` 模式。当使用内嵌 `ThreadedHTTPServer` 时，点击"停止"后 UI 显示已停止，但服务器线程仍在后台运行。

**文件**: `2workbench/presentation/ops/deploy/deploy_manager.py`

**修改方案**:

### 第一步：保存 server 实例

```python
# 修改 _start_service() 中的 run_server 内嵌函数
def _start_service(self) -> None:
    """启动 HTTP 服务"""
    port = self._port_spin.value()
    host = self._host_edit.text() or "127.0.0.1"

    try:
        script_path = Path(__file__).parent.parent.parent / "server_main.py"
        if not script_path.exists():
            from presentation.server import ThreadedHTTPServer, RequestHandler

            def run_server():
                self._http_server = ThreadedHTTPServer((host, port), RequestHandler)
                logger.info(f"HTTP 服务启动: http://{host}:{port}")
                self._http_server.serve_forever()

            self._server_thread = threading.Thread(target=run_server, daemon=True)
            self._server_thread.start()
        else:
            # ... subprocess 逻辑不变
```

### 第二步：在 `_stop_service` 中关闭 server

```python
def _stop_service(self) -> None:
    """停止 HTTP 服务"""
    # 停止子进程
    if self._server_process:
        self._server_process.terminate()
        try:
            self._server_process.wait(timeout=5)
        except subprocess.TimeoutExpired:
            self._server_process.kill()
        self._server_process = None

    # 停止内嵌服务器
    if hasattr(self, '_http_server') and self._http_server:
        self._http_server.shutdown()
        self._http_server = None

    self._deploy_status = "idle"
    self._status_label.setText("⏹ 已停止")
    self._status_label.setStyleSheet("font-weight: bold; font-size: 14px;")
    self._run_status.setText("已停止")
    self._btn_start.setEnabled(True)
    self._btn_stop.setEnabled(False)
    logger.info("服务已停止")
```

### 第三步：初始化 `_http_server`

```python
# 在 __init__ 中添加
def __init__(self, parent=None):
    super().__init__(parent)
    self._deploy_status = "idle"
    self._server_process = None
    self._server_thread = None
    self._http_server = None  # 新增
    self._setup_ui()
```

**验收标准**:
- [ ] 内嵌服务器模式下点击"停止"后，端口被释放
- [ ] 停止后可以重新启动服务
- [ ] 子进程模式仍然正常工作

## Step 2.2: 修复停止 Agent 后按钮状态不恢复

**问题**: BUG-022 — `_on_stop_agent` 使用 `QThread.terminate()` 强制终止线程，不会触发 `finished` 信号，导致运行/停止按钮状态不恢复。

**文件**: `2workbench/presentation/main_window.py` L1350-1357

**修改方案**:

```python
# 修改 _on_stop_agent()
def _on_stop_agent(self) -> None:
    """停止 Agent"""
    if hasattr(self, '_agent_thread') and self._agent_thread.isRunning():
        self._agent_thread.terminate()
        self._agent_thread.wait()
        # 手动恢复按钮状态（terminate 不触发 finished 信号）
        self._run_action.setEnabled(True)
        self._stop_action.setEnabled(False)
        self._show_message("Agent 已停止")
    else:
        self._show_message("Agent 未在运行")
```

**验收标准**:
- [ ] 点击"停止"后，运行按钮恢复为可用
- [ ] 点击"停止"后，停止按钮恢复为禁用
- [ ] 停止后可以再次点击"运行"
