## Step 2.1: 清理 project_selector.py Header/Footer 硬编码背景色

**问题**: L376 `#151515`（Header）和 L529 `#181818`（Footer）仍硬编码

**文件**: `2workbench/presentation/dialogs/project_selector.py`

**修改方案**:

在 `_apply_theme()` 方法中，将 Header 和 Footer 的背景色改为从主题获取：

```python
# 修改前
background-color: #151515;  # Header
background-color: #181818;  # Footer

# 修改后
background-color: {bg_darker};  # Header — 需要在 PALETTES 中新增或复用
background-color: {bg_darkest}; # Footer — 需要在 PALETTES 中新增或复用
```

如果 PALETTES 中没有 `bg_darker`/`bg_darkest`，可以复用 `bg_secondary` 或在 `theme/manager.py` 中新增：

```python
# theme/manager.py PALETTES 中新增
"dark": {
    ...
    "bg_darker": "#151515",   # Header 背景
    "bg_darkest": "#181818",  # Footer 背景
},
"light": {
    ...
    "bg_darker": "#e8e8e8",
    "bg_darkest": "#f0f0f0",
}
```

**验收标准**:
- [ ] Header/Footer 背景色跟随主题切换
- [ ] 无硬编码颜色残留

## Step 2.2: 清理 graph_editor.py 硬编码颜色

**问题**: L83 节点边框 `#3e3e42`、L91 标签文字 `#ffffff`、L103 类型标签 `#cccccc` 硬编码

**文件**: `2workbench/presentation/editor/graph_editor.py`

**修改方案**:

```python
# 修改前
QPen(QColor("#3e3e42"), 2)          # L83 节点边框
QColor("#ffffff")                    # L91 标签文字
QColor("#cccccc")                    # L103 类型标签

# 修改后
from presentation.theme.manager import theme_manager
p = theme_manager.PALETTES.get(theme_manager.current_theme, {})
QPen(QColor(p.get("border", "#3e3e42")), 2)
QColor(p.get("text_bright", "#ffffff"))
QColor(p.get("text_primary", "#cccccc"))
```

**验收标准**:
- [ ] 节点颜色跟随主题切换
- [ ] 无硬编码颜色残留

## Step 2.3: 修复 graph_editor.py 类变量计数器冲突

**问题**: L539 `_node_counter = 0` 是类变量，多个实例共享

**文件**: `2workbench/presentation/editor/graph_editor.py`

**修改方案**:

```python
# 修改前（类变量）
class GraphScene(QGraphicsScene):
    _node_counter = 0

# 修改后（实例变量）
class GraphScene(QGraphicsScene):
    def __init__(self, parent=None):
        super().__init__(parent)
        self._node_counter = 0  # 实例变量
```

同时确保所有引用 `GraphScene._node_counter` 的地方改为 `self._node_counter`。

**验收标准**:
- [ ] 多个图编辑器实例的节点 ID 不冲突

## Step 2.4: 统一硬编码字体

**涉及文件**:
- `project_selector.py` L85/L93/L100: `QFont("Microsoft YaHei", ...)`
- `main_window.py` L1106: `QFont("Cascadia Code", ...)`
- `runtime_panel.py` L67-70: `font-family` 硬编码

**修改方案**:

在 `theme/manager.py` 中新增字体变量：

```python
PALETTES = {
    "dark": {
        ...
        "font_family": '"Microsoft YaHei", "PingFang SC", "Noto Sans CJK SC", sans-serif',
        "mono_font": '"Cascadia Code", "Consolas", "Monaco", "Courier New", monospace',
    },
    "light": {
        ...
        "font_family": '"Microsoft YaHei", "PingFang SC", "Noto Sans CJK SC", sans-serif',
        "mono_font": '"Cascadia Code", "Consolas", "Monaco", "Courier New", monospace',
    }
}
```

然后在各文件中引用：

```python
from presentation.theme.manager import theme_manager
p = theme_manager.PALETTES.get(theme_manager.current_theme, {})
font = QFont(p.get("font_family", "sans-serif"), 14)
mono = p.get("mono_font", "monospace")
```

**验收标准**:
- [ ] 无硬编码字体名残留
- [ ] 跨平台字体正确显示
