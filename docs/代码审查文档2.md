## 二、严重 Bug 列表

### BUG-014（新）: prompt_editor.py self._logger 不存在

**文件**: `editor/prompt_editor.py` 第 234 行
**严重性**: 🔴 致命 — 保存 Prompt 时 AttributeError

```python
self._logger.info("保存 Prompt: ...")
```

**问题**: 模块级别定义的是 `logger`，`self._logger` 不存在。这是修复其他问题时引入的**回归 BUG**。

**修复**: 将 `self._logger` 改为 `logger`

---

### BUG-015（新）: runtime_panel.py QLabel 整数运算

**文件**: `ops/debugger/runtime_panel.py` 第 113 行
**严重性**: 🔴 致命 — 流式输出时 TypeError

```python
self._line_count += token.count('\n')
```

**问题**: `self._line_count` 是一个 `QLabel` 对象，不能直接 `+=` 整数。

**修复**:
```python
self._line_count_int += token.count('\n')
self._line_count.setText(f"行数: {self._line_count_int}")
```

---

### BUG-016（新）: runtime_panel.py _update_stats 方法不存在

**文件**: `ops/debugger/runtime_panel.py` 第 114 行
**严重性**: 🔴 致命 — 流式输出时 AttributeError

```python
self._update_stats()
```

**问题**: `ConsoleOutput` 类中没有定义 `_update_stats()` 方法。

**修复**: 在 `ConsoleOutput` 中添加 `_update_stats()` 方法，或移除此调用。
