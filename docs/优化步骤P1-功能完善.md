## Step 1.1: 修复内嵌服务器线程无法停止

**问题**: BUG-021 — `_stop_service` 只处理 subprocess 模式，未处理 threading 模式。

**文件**: `2workbench/presentation/ops/deploy/deploy_manager.py`

**修改方案**：

1. `__init__` 中添加 `self._http_server = None`

2. `_start_service` 中保存 server 实例：
```python
def run_server():
    self._http_server = ThreadedHTTPServer((host, port), RequestHandler)
    logger.info(f"HTTP 服务启动: http://{host}:{port}")
    self._http_server.serve_forever()
```

3. `_stop_service` 中添加线程停止逻辑：
```python
def _stop_service(self) -> None:
    # 停止子进程
    if self._server_process:
        self._server_process.terminate()
        try:
            self._server_process.wait(timeout=5)
        except subprocess.TimeoutExpired:
            self._server_process.kill()
        self._server_process = None

    # 停止内嵌服务器
    if hasattr(self, '_http_server') and self._http_server:
        self._http_server.shutdown()
        self._http_server = None

    self._deploy_status = "idle"
    # ... 其余 UI 更新
```

**验收标准**:
- [ ] 内嵌服务器模式下点击停止后端口被释放
- [ ] 子进程模式仍然正常工作

## Step 1.2: 修复停止 Agent 后按钮状态不恢复

**问题**: BUG-022 — `_on_stop_agent` 使用 `QThread.terminate()` 不触发 `finished` 信号。

**文件**: `2workbench/presentation/main_window.py` L1350-1357

**修改方案**：
```python
def _on_stop_agent(self) -> None:
    if hasattr(self, '_agent_thread') and self._agent_thread.isRunning():
        self._agent_thread.terminate()
        self._agent_thread.wait()
        # 手动恢复按钮状态
        self._run_action.setEnabled(True)
        self._stop_action.setEnabled(False)
        self._show_message("Agent 已停止")
    else:
        self._show_message("Agent 未在运行")
```

**验收标准**:
- [ ] 停止后运行按钮恢复可用
- [ ] 停止后可再次运行 Agent

## Step 1.4: 修复第一个标签页无法关闭

**问题**: BUG-028 — `_on_close_current_tab` 中 `if index > 0` 导致第一个标签页无法关闭。

**文件**: `2workbench/presentation/main_window.py` L1210-1214

**修改方案**：
```python
def _on_close_current_tab(self) -> None:
    index = self.center_panel.tab_widget.currentIndex()
    if index >= 0:
        self.center_panel._on_tab_close(index)
```

**验收标准**:
- [ ] Ctrl+W 可以关闭任意标签页
- [ ] 最后一个标签页不被关闭（由 CenterPanel._on_tab_close 保护）
