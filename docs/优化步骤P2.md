## Step 2.1: 清理无效的 statusBar().showMessage() 调用

**问题**: BUG-012 — 状态栏已隐藏，10+ 处 showMessage 调用无效

**文件**: `2workbench/presentation/main_window.py` 多处

**修改方案**:

将所有 `self.statusBar().showMessage(msg, duration)` 替换为工具栏状态标签更新：

```python
# 定义统一消息方法（在 MainWindow 中）
def _show_message(self, msg: str, duration: int = 3000) -> None:
    """统一消息显示 — 使用工具栏状态标签"""
    self._toolbar_status_agent.setText(f"💬 {msg}")
    if duration > 0:
        QTimer.singleShot(duration, lambda: self._toolbar_status_agent.setText("🤖 空闲"))

# 全局替换
# 旧: self.statusBar().showMessage("项目已保存", 2000)
# 新: self._show_message("项目已保存", 2000)
```

涉及行号：830, 855, 883, 889, 897, 926, 929, 979, 983, 1028

**验收标准**:
- [ ] 所有操作反馈通过工具栏状态标签显示
- [ ] 无残留的 statusBar().showMessage() 调用

## Step 2.7: 抽取项目加载逻辑

**问题**: UX-006 — 项目加载代码重复 4 次

**文件**: `2workbench/presentation/main_window.py`

**修改方案**:

```python
def _load_project_to_editors(self) -> None:
    """将当前项目加载到所有编辑器（统一入口）"""
    from presentation.project.manager import project_manager
    self.center_panel.remove_welcome_tab()

    graph = project_manager.load_graph()
    self.center_panel.show_graph_editor(graph)

    prompts = {}
    for name in project_manager.list_prompts():
        prompts[name] = project_manager.load_prompt(name)
    self.center_panel.show_prompt_editor(prompts)

    self.center_panel.show_tool_manager(self.right_panel)
    self.left_panel.setVisible(True)
    self.left_panel.load_project_tree(str(project_manager._project_path))
```

**验收标准**:
- [ ] 4 处项目加载逻辑统一为 1 个方法
