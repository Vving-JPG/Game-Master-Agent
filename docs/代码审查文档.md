## 一、项目概述

Game Master Agent IDE 是一个基于 **PyQt6** 的桌面端集成开发环境，采用四层架构（Foundation / Core / Feature / Presentation），用于创建和调试基于 LangGraph 的游戏主持人 AI Agent。

表现层（P4 Presentation）包含以下模块：

| 模块 | 文件数 | 核心职责 |
|------|--------|---------|
| 主窗口 | 1 | 三栏布局 IDE、菜单栏、工具栏 |
| 对话框 | 3 | 项目选择器、设置对话框、新建项目 |
| 编辑器 | 3 | 图编辑器、Prompt 管理器、工具管理器 |
| Ops 面板 | 7 | 调试器、评估、知识库、安全、编排、日志、部署 |
| 通用组件 | 3 | BaseWidget、StyledButton、SearchBar |
| 主题系统 | 3 | dark.qss、light.qss、ThemeManager |
| HTTP 服务 | 2 | server.py、state_api.py |
| 项目管理 | 2 | manager.py、new_dialog.py |

### BUG-006: graph_editor.py 边删除逻辑错误（残留）

**文件**: `editor/graph_editor.py` 第 318-330 行
**严重性**: 🟠 高 — 删除节点后边残留
**状态**: ⚠️ **部分修复** — 边从列表移除但未从场景移除

```python
def remove_node(self, node_id: str) -> None:
    node = self._nodes.pop(node_id, None)
    if node:
        self._edges = [
            e for e in self._edges
            if e.source.node_id != node_id and e.target.node_id != node_id
        ]
        for edge in self._edges[:]:  # 遍历已过滤的列表
            if edge.scene():
                pass  # ← 什么都不做！
        self.removeItem(node)
```

**残留问题**: 被删除节点的边从 `self._edges` 列表中移除了，但**没有调用 `self.removeItem(edge)` 从 QGraphicsScene 中移除**，导致边仍然可见且占用内存。
