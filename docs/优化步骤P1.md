## Step 1.1: 修复 runtime_panel.py self._count_label 未定义

**问题**: BUG-017 — 流式输出时 AttributeError

**文件**: `2workbench/presentation/ops/debugger/runtime_panel.py` 第 130 行

**根因**: `_update_stats()` 引用了 `self._count_label`，但 `_setup_ui` 中从未创建该 QLabel。

**修改方案**:

在 `_setup_ui` 的统计标签区域添加：

```python
# 在创建 self._line_count 附近添加
self._count_label = QLabel("字符: 0")
self._count_label.setStyleSheet("color: #858585; font-size: 11px;")
stats_layout.addWidget(self._count_label)
```

**验收标准**:
- [ ] 流式输出时字符数正确更新
- [ ] 不抛出 AttributeError
