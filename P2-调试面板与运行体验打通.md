## 2.1 调试面板运行控制连接 GMAgent

**文件**：`2workbench/presentation/ops/debugger/runtime_panel.py` + `main_window.py`

**当前问题**：
调试面板的运行/暂停/停止按钮只发出 EventBus 事件（`ui.debugger.run`），没有任何组件消费这些事件。调试面板是一个"孤岛"。

**修复方案**：在 `MainWindow` 中订阅调试面板事件，转发到 GMAgent。

```python
# main_window.py — 在 __init__ 或 setup_connections 中添加
def _setup_debugger_connections(self):
    """连接调试面板到 Agent"""
    event_bus.subscribe("ui.debugger.run", self._on_debugger_run)
    event_bus.subscribe("ui.debugger.stop", self._on_debugger_stop)
    event_bus.subscribe("ui.debugger.input", self._on_debugger_input)

def _on_debugger_run(self, event):
    """调试面板触发运行"""
    # 复用现有的 _on_run_agent 逻辑
    self._on_run_agent()

def _on_debugger_stop(self, event):
    """调试面板触发停止"""
    self._on_stop_agent()

def _on_debugger_input(self, event):
    """调试面板发送用户输入"""
    text = event.get("text", "")
    if not text or not hasattr(self, '_current_agent') or not self._current_agent:
        return
    # 在后台线程中运行 Agent
    # ... 复用 AgentThread 逻辑 ...
```

## 2.2 变量监视器实时刷新

**文件**：`2workbench/presentation/ops/debugger/runtime_panel.py`

**当前问题**：
`VariableWatcher.refresh()` 只发出 `ui.debugger.request_state` 事件，没有消费者提供实际状态数据。

**修复**：

```python
# runtime_panel.py — VariableWatcher
def refresh(self):
    """刷新变量监视器"""
    if not hasattr(self, '_table'):
        return
    self._table.setRowCount(0)

    # 直接从 GMAgent 获取状态快照
    try:
        from presentation.main_window import MainWindow
        # 通过 event_bus 获取最近的 Agent 状态
        # 或直接访问 main_window._current_agent
    except ImportError:
        pass

# 更好的方案：在 MainWindow 中订阅 request_state 事件
# main_window.py
event_bus.subscribe("ui.debugger.request_state", self._on_request_state)

def _on_request_state(self, event):
    """响应调试面板的状态请求"""
    if not hasattr(self, '_current_agent') or not self._current_agent:
        return
    snapshot = self._current_agent.get_state_snapshot()
    event_bus.emit("ui.debugger.state_update", {"state": snapshot})
```

## 2.4 Agent 运行结果展示优化

**文件**：`2workbench/presentation/main_window.py`

**当前**：Agent 运行结果通过 `QMessageBox` 弹窗显示，不够友好。

**改进**：将结果流式展示到调试面板的 ConsoleOutput。

```python
# main_window.py — AgentThread
class AgentThread(QThread):
    # 已有 signal_result
    # 新增 signal_token 用于流式输出
    token_received = Signal(str)

    def run(self):
        # ... 现有逻辑 ...
        # 在流式输出时发出 token
        event_bus.subscribe("feature.ai.stream.token", self._on_token)

    def _on_token(self, event):
        text = event.get("content", "")
        if text:
            self.token_received.emit(text)
```
