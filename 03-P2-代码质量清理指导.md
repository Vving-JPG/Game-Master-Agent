## 二、Bug 修复

### 2.1 EventBus target 过滤失效

**文件**：`foundation/event_bus.py` 第196行

**问题**：`__qualname__` 匹配逻辑几乎永远不生效。

```python
# 修改前
if event.target and sub.handler.__qualname__ != event.target:
    continue

# 修改后 — 改用 handler 的模块名匹配
if event.target and sub.handler.__module__ != event.target:
    continue
```

### 2.2 temperature=0 被错误忽略

**文件**：`foundation/llm/openai_client.py`

**问题**：`temperature or self._default_temperature` 中，`temperature=0` 被 `or` 视为 falsy。

```python
# 修改前（出现在 chat_async 和 stream 方法中）
temperature = temperature or self._default_temperature

# 修改后
temperature = temperature if temperature is not None else self._default_temperature
```

### 2.3 apply_template 浅拷贝风险

**文件**：`core/constants/npc_templates.py`

```python
# 修改前
def apply_template(template_name: str, overrides: dict | None = None) -> dict:
    template = get_template(template_name)
    result = dict(template)  # 浅拷贝
    if overrides:
        result.update(overrides)
    return result

# 修改后
import copy

def apply_template(template_name: str, overrides: dict | None = None) -> dict:
    template = get_template(template_name)
    result = copy.deepcopy(template)  # 深拷贝
    if overrides:
        result.update(overrides)
    return result
```

## 五、SQL Schema 增强

**文件**：`core/models/schema.sql`

添加 CHECK 约束，与 Python 枚举保持一致：

```sql
-- 示例：quests 表
CREATE TABLE IF NOT EXISTS quests (
    ...
    status TEXT NOT NULL DEFAULT 'not_started'
        CHECK (status IN ('not_started', 'active', 'completed', 'failed', 'abandoned')),
    quest_type TEXT NOT NULL DEFAULT 'main'
        CHECK (quest_type IN ('main', 'side', 'hidden')),
    ...
);

-- 示例：items 表
CREATE TABLE IF NOT EXISTS items (
    ...
    item_type TEXT NOT NULL DEFAULT 'misc'
        CHECK (item_type IN ('weapon', 'armor', 'potion', 'misc', 'key', 'quest')),
    rarity TEXT NOT NULL DEFAULT 'common'
        CHECK (rarity IN ('common', 'uncommon', 'rare', 'epic', 'legendary')),
    stackable INTEGER DEFAULT 0 CHECK (stackable IN (0, 1)),
    usable INTEGER DEFAULT 0 CHECK (usable IN (0, 1)),
    ...
);
```

## 六、Repository 代码去重

**文件**：`core/models/repository.py`

### 6.1 提取公共方法

```python
class BaseRepository:
    def _resolve_db_path(self, db_path: str | None) -> str:
        return db_path or self._db_path

    def _build_set_clause(self, kwargs: dict) -> tuple[str, list]:
        """构建 UPDATE SET 子句"""
        sets = []
        values = []
        for key, value in kwargs.items():
            if key in self._allowed_fields:  # 列名白名单
                sets.append(f"{key} = ?")
                values.append(value)
        return ", ".join(sets), values
```
