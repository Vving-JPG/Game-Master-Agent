## Step 2.2: 清理 main_window.py 硬编码颜色

**问题**: QUAL-002 — 工具栏状态标签、欢迎页、右侧面板等硬编码颜色

**文件**: `main_window.py`

**涉及行号**: L200, L328, L803-813, L840-849

**修改方案**:

```python
# 修改前
label.setStyleSheet("color: #858585;")

# 修改后：从主题管理器获取颜色
from presentation.theme.manager import theme_manager
palette = theme_manager.PALETTES.get(theme_manager.current_theme, {})
text_secondary = palette.get("text_secondary", "#858585")
label.setStyleSheet(f"color: {text_secondary};")
```

或更优方案：在 `ThemeManager.apply()` 中发射信号，让所有 widget 刷新。

**验收标准**:
- [ ] Dark 主题下颜色正确
- [ ] Light 主题下颜色正确
- [ ] 切换主题后即时更新

## Step 2.3: 清理 Ops 面板硬编码颜色

**涉及文件及行号**:

| 文件 | 行号 | 硬编码值 |
|------|------|---------|
| `event_monitor.py` | L57, L115-119 | `#858585`, `#cccccc`, `#f44747`, `#4ec9b0`, `#569cd6`, `#dcdcaa`, `#6e6e6e` |
| `runtime_panel.py` | L57, L66-68, L78, L248 | `#858585`, 字体样式 |
| `knowledge_editor.py` | L394, L400 | `#858585` |
| `eval_workbench.py` | L91 | `#858585` |
| `deploy_manager.py` | L46, L97 | 样式硬编码 |

**修改方案**:

统一使用主题变量替换。建议在 `theme/manager.py` 的 `PALETTES` 中定义语义化颜色变量：

```python
PALETTES = {
    "dark": {
        ...
        "text_secondary": "#858585",
        "text_success": "#4ec9b0",
        "text_error": "#f44747",
        "text_warning": "#dcdcaa",
        "text_info": "#569cd6",
        "text_disabled": "#6e6e6e",
    },
    "light": {
        ...
        "text_secondary": "#6a6a6a",
        "text_success": "#2e7d32",
        "text_error": "#d32f2f",
        "text_warning": "#f57f17",
        "text_info": "#1565c0",
        "text_disabled": "#b0b0b0",
    }
}
```

然后在各文件中引用：

```python
from presentation.theme.manager import theme_manager
p = theme_manager.PALETTES.get(theme_manager.current_theme, {})
label.setStyleSheet(f"color: {p.get('text_secondary', '#858585')};")
```

**验收标准**:
- [ ] 所有 Ops 面板跟随主题切换
- [ ] 无硬编码颜色值残留

## Step 2.4: 清理 project_selector.py 硬编码样式

**问题**: ProjectCard 和 ProjectSelector 对话框大量硬编码暗色样式

**文件**: `project_selector.py` L37-60, L128-159

**修改方案**:

将 `ProjectCard` 和 `ProjectSelector` 的样式改为使用 `${variable}` 模板变量，通过 `theme_manager.apply_qss_template()` 处理。

```python

qss = theme_manager.apply_qss_template(PROJECT_SELECTOR_QSS)
self.setStyleSheet(qss)
```
