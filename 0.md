# AI监控面板为空问题排查记录

## 问题
管理端 AI 监控面板显示为空。

## 原因
GameMaster 的 `process_stream()` 方法（WebSocket 使用）没有调用 `metrics_repo.record_llm_call()` 记录 LLM 调用。

## 修复
在 `src/agent/game_master.py` 的 `process_stream()` 方法中添加记录代码：

```python
# 记录调用开始时间
start_time = time.time()
response = self.llm.chat_with_tools(messages, self.tools)
latency_ms = int((time.time() - start_time) * 1000)

# 记录 LLM 调用
usage = response.usage
tool_calls_count = len(message.tool_calls) if message.tool_calls else 0
tool_names = [tc.function.name for tc in message.tool_calls] if message.tool_calls else []

metrics_repo.record_llm_call(
    world_id=self.world_id,
    call_type="chat_with_tools",
    prompt_tokens=usage.prompt_tokens,
    completion_tokens=usage.completion_tokens,
    latency_ms=latency_ms,
    tool_calls_count=tool_calls_count,
    tool_names=tool_names,
    error="",
    db_path=self.db_path
)
```

## 验证
修复后 AI 监控面板正常显示：
- 总调用次数: 4
- 总 Prompt Tokens: 7725
- 总 Completion Tokens: 1621
- 平均延迟: 约 3-4 秒
