## 二、严重 Bug 列表

### BUG-017（新）: runtime_panel.py self._count_label 未定义

**文件**: `ops/debugger/runtime_panel.py` 第 130 行
**严重性**: 🔴 致命 — 流式输出时 AttributeError

```python
def _update_stats(self) -> None:
    self._count_label.setText(f"字符: {self._count}")  # _count_label 未定义！
    self._line_count.setText(f"行数: {self._line_count_int}")
```

**修复**: 在 `_setup_ui` 中添加 `self._count_label = QLabel("字符: 0")` 并布局。

---

### BUG-018（新）: tool_manager.py self._logger 未定义

**文件**: `editor/tool_manager.py` 第 395/415/417 行
**严重性**: 🔴 致命 — 添加自定义工具时 AttributeError

```python
self._logger.info(f"自定义工具添加: {name}")       # L395
self._logger.info(f"工具已注册到 Agent: {tool.name}") # L415
self._logger.error(f"工具注册到 Agent 失败: {e}")    # L417
```

**修复**: 将 `self._logger` 改为 `logger`。
