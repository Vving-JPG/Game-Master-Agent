## 修复 1：Feature 层移除 PyQt6 依赖

**文件**：`feature/services/api_tester.py`

**问题**：`ApiTestWorker` 继承 `QThread`，使用 `pyqtSignal`，属于 Presentation 层代码。

**修复方案**：

```python
# 修改前（feature/services/api_tester.py）
from PyQt6.QtCore import QThread, pyqtSignal

class ApiTestWorker(QThread):
    finished = pyqtSignal(object)  # TestResult
    error = pyqtSignal(str)

    def run(self):
        # ... 测试逻辑

# 修改后
import threading
from typing import Callable

class ApiTestWorker(threading.Thread):
    def __init__(self, config: dict, on_result: Callable, on_error: Callable):
        super().__init__(daemon=True)
        self._config = config
        self._on_result = on_result
        self._on_error = on_error

    def run(self):
        try:
            result = asyncio.run(self._do_test(self._config))
            self._on_result(result)
        except Exception as e:
            self._on_error(str(e))
```

**同步修改**：
- `feature/services/__init__.py`：移除 `ApiTestWorker` 导出（或保留新版本）
- `presentation/dialogs/settings_dialog.py`：将 `pyqtSignal` 连接移到 Presentation 层

```python
# presentation/dialogs/settings_dialog.py 中
# 修改前
worker = ApiTestWorker(config)
worker.finished.connect(self._on_test_result)

# 修改后
worker = ApiTestWorker(config, on_result=self._on_test_result, on_error=self._on_error)
worker.start()
```

**修复方案**：

```python
# 修改前
def init_db(db_path: str | None = None):
    # ...

# 修改后 — schema 路径参数化
```

```json
      "weight": 5
```

**步骤 2**：修改 `model_router.py` 从配置文件加载

```python
# 修改后
def _load_rules(self) -> list[dict]:
    """从配置文件加载路由规则，不再硬编码业务关键词"""
    rules_path = Path(__file__).parent.parent / "config" / "model_rules.json"
    if rules_path.exists():
        with open(rules_path, 'r', encoding='utf-8') as f:
            data = json.load(f)
            return data.get("rules", [])
    return []  # 无配置时返回空规则，使用默认模型
```
