## Step 3.1: 实现部署服务启停

**问题**: `_start_service()` 和 `_stop_service()` 仅切换 UI 状态标签

**文件**: `2workbench/presentation/ops/deploy/deploy_manager.py`

**修改方案**:

```python
import subprocess
import signal
import sys

class DeployManager(QWidget):
    def __init__(self):
        ...
        self._server_process = None

    def _start_service(self) -> None:
        """启动 HTTP 服务"""
        port = self._port_spin.value()
        host = self._host_edit.text() or "127.0.0.1"  # 默认 localhost 更安全

        try:
            # 启动子进程运行 server
            script_path = Path(__file__).parent.parent.parent / "server_main.py"
            if not script_path.exists():
                # 如果没有独立入口，使用内嵌服务器
                from presentation.server import start_server
                self._server_thread = threading.Thread(
                    target=start_server, args=(host, port), daemon=True
                )
                self._server_thread.start()
            else:
                self._server_process = subprocess.Popen(
                    [sys.executable, str(script_path), "--host", host, "--port", str(port)],
                    stdout=subprocess.PIPE,
                    stderr=subprocess.PIPE,
                )

            self._status_label.setText(f"✅ 运行中 — http://{host}:{port}")
            self._status_label.setStyleSheet("color: #4ec9b0;")
            self._start_btn.setEnabled(False)
            self._stop_btn.setEnabled(True)
            logger.info(f"服务已启动: http://{host}:{port}")
        except Exception as e:
            logger.error(f"启动服务失败: {e}")
            QMessageBox.critical(self, "启动失败", str(e))

    def _stop_service(self) -> None:
        """停止 HTTP 服务"""
        if self._server_process:
            self._server_process.terminate()
            self._server_process.wait(timeout=5)
            self._server_process = None

        self._status_label.setText("⏹ 已停止")
        self._status_label.setStyleSheet("color: #858585;")
        self._start_btn.setEnabled(True)
        self._stop_btn.setEnabled(False)
        logger.info("服务已停止")
```

**验收标准**:
- [ ] 点击启动后服务实际运行
- [ ] 点击停止后服务正确终止
- [ ] 状态标签实时更新

## Step 3.2: 实现文件编辑保存

**问题**: F-002 — 双击文件打开后为只读

**文件**: `2workbench/presentation/main_window.py`

**修改方案**:

在 `_open_text_editor` 中将编辑器设为可编辑，并绑定 Ctrl+S：

```python
def _open_text_editor(self, file_path: str) -> None:
    path = Path(file_path)
    if not path.exists():
        self._show_message(f"文件不存在: {file_path}")
        return

    try:
        content = path.read_text(encoding="utf-8")
    except Exception as e:
        self._show_message(f"读取失败: {e}")
        return

    editor = QTextEdit()
    editor.setPlainText(content)
    editor.setReadOnly(False)  # 可编辑

    # Ctrl+S 保存
    def save_file():
        try:
            path.write_text(editor.toPlainText(), encoding="utf-8")
            self._show_message(f"已保存: {path.name}", 2000)
            logger.info(f"文件已保存: {path}")
        except Exception as e:
            QMessageBox.critical(self, "保存失败", str(e))

    QShortcut(QKeySequence.StandardKey.Save, editor).activated.connect(save_file)

    tab_name = path.name
    index = self.center_panel.tab_widget.addTab(editor, tab_name)
    self.center_panel.tab_widget.setCurrentIndex(index)
    self.center_panel.remove_welcome_tab()
```

**验收标准**:
- [ ] 双击文件可编辑内容
- [ ] Ctrl+S 保存到原文件
- [ ] 保存成功有消息提示

## Step 3.3: 实现撤销/重做

**问题**: F-003 — Ctrl+Z/Y 被空菜单项拦截

**文件**: `2workbench/presentation/main_window.py`

**修改方案**:

QTextEdit 自带撤销/重做，只需移除拦截：

```python
# 在 _setup_menu 中，移除或禁用空的撤销/重做 action
# 方案 A：直接删除这两个 action
# 方案 B：连接到当前编辑器的 undo/redo

def _setup_menu(self):
    ...
    edit_menu = menubar.addMenu("编辑(&E)")

    undo_action = QAction("撤销(&Z)", self)
    undo_action.setShortcut("Ctrl+Z")
    undo_action.triggered.connect(self._on_undo)
    edit_menu.addAction(undo_action)

    redo_action = QAction("重做(&Y)", self)
    redo_action.setShortcut("Ctrl+Y")
    redo_action.triggered.connect(self._on_redo)
    edit_menu.addAction(redo_action)

def _on_undo(self) -> None:
    """撤销"""
    widget = self.center_panel.tab_widget.currentWidget()
    if isinstance(widget, QTextEdit):
        widget.undo()

def _on_redo(self) -> None:
    """重做"""
    widget = self.center_panel.tab_widget.currentWidget()
    if isinstance(widget, QTextEdit):
        widget.redo()
```

**验收标准**:
- [ ] Ctrl+Z 撤销文本编辑
- [ ] Ctrl+Y 重做文本编辑
